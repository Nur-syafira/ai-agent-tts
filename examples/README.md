# Примеры использования

Эта папка содержит рабочие примеры использования платформы.

## Базовые примеры

### `basic_dialog.py`

Минимальный пример обработки диалога:

```bash
uv run python examples/basic_dialog.py
```

Показывает:
- Инициализацию Policy Engine
- Обработку сообщений пользователя
- Получение ответов агента
- Извлечение структурированных данных (слотов)

### `custom_fsm.py`

Пример создания кастомного FSM для своего домена:

```bash
uv run python examples/custom_fsm.py
```

Показывает:
- Определение кастомных состояний
- Построение графа переходов
- Условные переходы

### `custom_prompts.py`

Пример настройки промптов под свою задачу:

```bash
uv run python examples/custom_prompts.py
```

Показывает:
- Кастомизацию system prompt
- Настройку промптов для извлечения слотов
- Настройку промптов для генерации ответов

## Требования

Перед запуском примеров убедитесь, что:

1. Все сервисы запущены:
   ```bash
   docker-compose up -d redis
   # Запустить vLLM, ASR Gateway, TTS Gateway, Policy Engine
   ```

2. Модели скачаны:
   ```bash
   uv run python scripts/download_models.py
   ```

3. Конфигурация настроена:
   ```bash
   cp .env.example .env
   # Отредактировать .env
   ```

## Расширение примеров

Вы можете использовать эти примеры как основу для своих экспериментов:

- Измените FSM под свой домен
- Настройте промпты под свою задачу
- Добавьте свои слоты в `DialogSlots`
- Интегрируйте с вашей телефонией/API
